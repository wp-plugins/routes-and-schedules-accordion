Lets transit agencies add a dynamically-created jQuery accordion for their routes and schedules information using a shortcode.

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. . Activate the plugin through the 'Plugins' menu in WordPress
3. Place the `[routes_and_schedules_accordion]` shortcode in your page(s) or post(s)
